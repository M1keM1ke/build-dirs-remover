# Build Dirs Remover

![Alt текст](assets/build-dirs-remover-icon.png)

### Allows you to remove folders that appear as a result of the assembly systems

## Supported assembly systems
- Maven
- Gradle

## Features
- Theme changing
- Language changing
- Path/dir to remove saving for any assembly system
